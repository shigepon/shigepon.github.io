DjangoでDateTimeField received a naive datetimeってWarningが出た

Date 土 22 11月 2014 Tags django / python 
RuntimeWarning: DateTimeField received a naive datetime

ってWarningが出るのに対する対処

# settings.py
USE_TZ = False

を書けばおk。要はタイムゾーン入れてくれってエラーなので、そんなものはない！と指定すればいいっぽい

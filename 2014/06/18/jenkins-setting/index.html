Ubuntu12.04でjenkinsのインストールと設定

Date 水 18 6月 2014 Tags linux / test1st

Ubuntu 12.04 LTS に Jenkins をインストール - MELODIC-Xを参考にして設定した。 Jenkins本体でアップグレード可能らしいので、aptitudeでインストール

$ sudo aptitude install jenkins

Nginx設定

location /jenkins {
    proxy_pass http://127.0.0.1:8080;
}

で、nginxは起動してたので再読み込み

$ sudo service nginx reload

Jenkins設定ファイル修正

jenkinsの設定ファイルは/etc/default/jenkinsにあるので編集 最下行の起動パラメーターに

--prefix=/jenkins

を追加。jenkins再起動

$ sudo service jenkins restart
